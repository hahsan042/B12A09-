## Project Overview

**GameHub** is a modern, responsive web application designed for gamers. It provides an interactive platform to explore, discover, and track popular games. Users can view game details, ratings, and download links, as well as manage their profile through secure authentication.

---

## Purpose

The purpose of this project is to create a **comprehensive gaming platform** where users can:  
- Explore the latest and popular games.  
- Access detailed information about each game.  
- Log in using email/password or Google authentication.  
- Update their profile information.  
- Subscribe to newsletters for updates and promotions.  
- Access protected routes securely.

---

## Key Features

- **Authentication:** Email/password login & registration, Google Sign-In.  
- **Protected Routes:** Game details are only accessible to logged-in users.  
- **Dynamic Tab Titles:** Page titles change dynamically based on the route.  
- **Homepage Sections:**  
  - Banner Slider (animated)  
  - Popular Games Section (cards with ratings)  
  - Newsletter subscription section  
- **Profile Management:** Users can update their name and photo URL.  
- **Responsive Design:** Fully mobile-friendly with dropdown menus.  
- **Animations:** Implemented with **Framer Motion** for smooth transitions and hover effects.  
- **404 Page:** User-friendly Not Found page for unknown routes.
